데이터마이닝(빅데이터에 Ai를 적용시키는 방법)
    빅데이터에서 유용한 정보를 추출하는 과정
    잠재적으로 활용가치가 있는 정보
    정보를 찾아내는 방법
        비즈니스 문제를 이해하고 해결하기 위한 정보를 찾아야 함

뉴런모델(다입력, 1출력)
    입력값 X가 여러개, Weight W는 X개수+1(상수)
    step func = activate func = 결정함수
        1) 계단 함수
            불연속적 : x가 0일 때 y값이 2개

        2) 임계함수
            연속이지만 한개의 함수값으로 표현 불가능
            x값의 범위에 따라 함수 다름

        3) 시그모이드 함수
            연속이며 모든 점에서 미분가능
    -> XOR 값 계산 못함
    -> MLP 진행, tanh, ReLU

순방향 전파
    가중치는 랜덤으로 설정
    학습결과 나온 결과를 비교하여 이전 가중치와 현재 가중치 중 가중치를 줄이는 방향으로 나감 = 엔트로피 역할

역전파
    가중치를 역순으로 보정하면서 다시 학습시키는 방법
    cost = loss = error = gap
    Adam, GD, XGD(mse 사용)
    loss func : mse, sse
